How to Use

Most styles of forms share the same html and classes with one major difference, the style class. Choose between any form styles with the .style-{#} class. A couple of notes:

  1. Form elements need to be grouped inside of .form-group class.
  2. For .style-2 the label must come before the input tag.
  3. To add more spacing to the left, use bootstraps .pl-{#} class. CrossWay extends this function to 10.
  4. Add alignment classes to the parent or to individual .item class to align to center, left, or right.
<div class="form style-1">
  <div class="row">
    <div class="col-md-4">
      <div class="card">
        <div class="icon font-3x">
        <i class="ti-gallery"></i>
      </div>
      <div class="content">
        <h4>Title</h4>
        <p>Description</p>
        <p class="read-more"><a href="#">Read More</a></p>
      </div>
    </div>
  </div>
</div>
