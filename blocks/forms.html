How to Use

Form styles differ in multiple ways. Check the source code to see how each blocks are created. In some cases, labels are not used. Like other blocks, parent class controls the overall style. Few notes:

  1. Form elements need to be grouped inside of .form-group class.
  2. For .style-2 the label must come before the input tag.
  3. The select, checkbox, radio all require a container to change it's default style.
<div class="form style-1">
  <div class="row">
    <div class="col-md-4">
      <div class="form-group">
        <label for="name"> Full Name </div>
        <input type="text" name="full-name" id="full-name" placeholder="John Doe">
      </div>
    </div>
  </div>
</div>
