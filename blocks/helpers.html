Class Helpers

Class helpers are created to add style easier without having to touch any css.

Basic Helpers

These basic helpers can make your coding life easier. These classes carry priority (!important value) over other elements.

Displays
<div class="display-{value}">CrossWay</div>
  • display-none
  • display-block
  • display-inline
  • display-inline-block
  • display-table
  • display-flex

You can also use Bootstrap's d-{value} classhelper.

Positions
<div class="position-{value}">CrossWay</div>
  • position-relative
  • position-absolute
  • position-static
  • position-fixed
  • position-inherit
  • position-initial

You can also use Bootstrap's fixed-top fixed-bottom sticky-top classhelper.

Extended Paddings
<div class="p{direction}-{#}">CrossWay</div>
  • .ptb-{#} - Adds top and bottom padding. Number values 1-3
  • .pt-{#} - Extends bootstraps padding top. Number values 1-10
  • .pb-{#} - Extends bootstraps padding bottom. Number values 1-10
  • .pl-{#} - Extends bootstraps padding left. Number values 1-10
  • .pr-{#} - Extends bootstraps padding right. Number values 1-10
Extended Margins
<div class="m{direction}-{#}">CrossWay</div>
  • .mtb-{#} - Adds top and bottom margin. Number values 1-3
  • .mt-{#} - Extends bootstraps margin top. Number values 1-10
  • .mb-{#} - Extends bootstraps margin bottom. Number values 1-10
  • .ml-{#} - Extends bootstraps margin left. Number values 1-10
  • .mr-{#} - Extends bootstraps margin right. Number values 1-10
Font Options
<div class="font-{format}-{value}">CrossWay</div>
  • .font-{#}x - Multiplies default font size by x amount. Number values 2x-5x
  • .font-weight-{value} - Changes font-weight value. Values: light, normal, bold.
  • .font-style-{value} - Changes font-style value. Values: italic, normal, oblique.
No- Removal Classes
<div class="no-{value}">CrossWay</div>
  • no-margin
  • no-margin-{direction}
  • no-padding
  • no-padding-{direction}
  • no-borders
  • no-style
Text Colors
<div class="text-{value}">CrossWay</div>
  • .text-light - Changes text color to light grey.
  • .font-{value} - Follows Bootstrap button color format.
  • .font-style-{value} - Changes font-style value. Values: italic, normal, oblique.
Object Sizes
<div class="w-{#}">CrossWay</div>
  • .w-{#} - Changes objects width based on % values. Example: w-25 = width: 25%.
  • .h-{#} - Changes objects height based on % values. Example: h-25 = height: 25%.
  • .vh-{#} - Changes object height based on vertical-height value (10 multipliers). Ex: vh-30 = height: 30vh.
Bootstrap 4 comes loaded with dozens of utility classes. These classes are very similar to classhelpers on CrossWay but does not overwrite each other.
