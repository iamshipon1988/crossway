How to Use

All service items share the same html and classes with one major difference, the style class. You have a choice of 4 different styles that you can control with the .style-{#} class. A couple of notes:

  1. Font-sizes are controlled by .font-{#}x (ex: .font-3x outputs 3x the default font size).
  2. Read More button has a parent class: read-more.
  3. To add more spacing to the left, use bootstraps .pl-{#} class. CrossWay extends this function to 10.
<div class="item">
  <div class="icon font-3x">
    <i class="ti-gallery"></i>
  </div>
  <div class="content">
    <h4>Title</h4>
    <p>Description</p>
    <p class="read-more"><a href="#">Read More</a></p>
  </div>
</div>
